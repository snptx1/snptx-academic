Validation Methodology (8-Check Audit)

Validation uses an 8-check audit template. Core checks are applied across primary benchmarks, with modality-specific checks activated where relevant. The goal of this section is to show that reported benchmark numbers are paired with explicit audit controls rather than treated as isolated headline metrics.

Validation summary

What the audit covers

  • Core checks address split sensitivity, randomization effects, and uncertainty estimation.
  • Modality-specific checks address spatial leakage, fusion leakage, and targeted uncertainty workflows.
  • The CV summary table translates these controls into a cross-modal variance surface.
Check 1

k-Fold Cross-Validation

5-fold stratified CV with per-fold metrics and variance reporting. Ensures performance is not an artifact of a single favorable split.

Check 2

Multi-Seed Stability

Multi-seed reruns are used on representative workloads to estimate variance under initialization and ordering effects.

Check 3

Bootstrap CIs

Bootstrap confidence intervals are computed for selected tasks to quantify metric uncertainty.

Check 4

Spatial Stratification

For tissue data (Visium), splits prevent spatial autocorrelation from inflating performance. Adjacent spots never split across train/test.

Check 5

Fusion Holdout

PCA embeddings extracted from training set only. Test set projected using frozen training-set PCA parameters. No data leakage.

Check 6

Conformal Prediction

Conformal methods are integrated for uncertainty analysis in targeted pipelines; broader rollout is ongoing.

Check 7

Learning Curves

Training set subsampled at [0.1, 0.25, 0.5, 0.75, 1.0] fractions to confirm models are not in data-starved regime.

Check 8

Trivial Baselines

Every model compared against majority-class and random baselines. All results significantly exceed trivial baselines.

Audit logic

Why the checks are mixed

The audit combines universal checks with modality-triggered controls because not every benchmark exposes the same leakage or uncertainty failure modes.

Evidence posture

What the table is meant to show

The CV summary gives the reader a compact view of score stability and variance across the reported modalities instead of relying only on single-point performance values.
