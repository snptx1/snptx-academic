Reproducibility controls

Determinism controls and boundaries

Fixed seeds: Global seed (1337) for all random operations (numpy, torch, sklearn).

Deterministic CUDA controls: torch.use_deterministic_algorithms(True) where supported.

Versioned configuration: Every run defined by YAML committed to version control.

DAG-enforced ordering: Snakemake enforces explicit dependency order with no implicit stage coupling.

Artifact hashing: DVC checksum workflows are configured; full end-to-end enforcement across the primary pipeline remains in progress.
